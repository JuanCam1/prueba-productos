Necesitas:

- [Node.js 18+ ](https://nodejs.org/en/).
- [Git](https://git-scm.com/).

1. [Fork](https://github.com/JuanCam1/prueba-productos.git) de este repositorio y clonarlo localmente:

2. Instala las dependencias:

```bash
bun install
```

```bash
bun run dev
```
